# TDS Photography — Quotation Generator

> A fast, professional tool for generating photography quotations — built exclusively for Rehan Tank of TDS Photography.

---

## What is this?

TDS Photography works with architectural and interior clients across Mumbai, Pune, and beyond. Every new project starts with a quotation — and this tool makes creating one effortless.

No more manually adjusting Canva templates. Just fill in the details, preview the quotation live, and download a clean PDF in seconds.

---

## How it works

**1. Fill in the details**
Enter the client's site name, shoot type, date, and add line items — each with a description, number of photos, reels/videos, and the price.

**2. See it live**
The quotation preview updates in real time on the right side — exactly how it'll look when printed.

**3. Download as PDF**
One click opens the print dialog. Save as PDF and it's ready to send to the client.

---

## What the quotation includes

- TDS Photography logo and branding
- "Quotation For" — client site name and shoot type
- Date
- Itemised table — Description / Photos / Reels/Video / Total
- Optional discount with original price shown as strikethrough
- Standard note about travel and accommodation
- Footer with brand links and contact details
